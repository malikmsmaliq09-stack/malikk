1. Instruksi Command Prompt (CMD)

CMD digunakan untuk berinteraksi dengan sistem Windows melalui baris perintah teks.

Perintah Fungsi Ringkasan
ipconfig Cek konfigurasi IP Menampilkan IP address perangkat saat ini.
ping [alamat] Tes koneksi Mengirim paket data untuk cek respon server.
cls Clear Screen Membersihkan riwayat teks di layar CMD.

3. Networking Dasar

IP Address: Identitas unik perangkat di jaringan.

Subnet Mask: Memisahkan Network ID dan Host ID.

Broadcast: Alamat untuk mengirim data ke semua host sekaligus.

Ringkasan: Tanpa ketiga hal ini, perangkat tidak bisa "mengobrol" dalam jaringan.
